# HE 김중태 글꼴
김중태 IT위원장의 폰트입니다.<br>
인터넷에 배포되고 있는 Neo둥근모와 둥근모꼴+Fixedsys와 달리 둥근모꼴의 라틴폰트는 김중태 원장의 영문둥근모꼴을 이용했습니다. 

## 폰트 지원범위
* ASCII + 한글11172자 영역을 지원합니다.

## 사용 도구
* [Pixel Font Maker](https://github.com/wintiger0222/pixel-font-maker) : 폰트 제작
* CrystalTile2

## 참조 폰트
* [둥근모꼴](https://ith.kr/about/font.html)
