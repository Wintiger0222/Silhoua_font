# HGViola15_Font
11x15 크기의 비트맵 폰트

## 폰트 지원범위
* 한글11172자 + ASCII 영역을 지원합니다.

## 사용 도구
* [Pixel Font Maker](https://github.com/exqt/pixel-font-maker) : 폰트 제작
* CrystalTile2

### 참조 폰트
* 대우 아이큐2000 BIOS 폰트

## 참조 프로젝트
 * [galmuri 폰트 프로젝트](https://github.com/quiple/galmuri)
